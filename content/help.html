FFUZBL Help Page

About

FFUZBL provides using layout of UZBL browser that adheres to the Unix's design philosophy.You can switch to UZBL Mode by pressing F9 button in the keyboard.As you will expect, F9 button can be used for switching back to classical Firefox design when ffuzbl is enabled.

While you're in FFUZBL Mode, you'll need some extra shortcuts. That's why FFUZBL has some special shortcuts.For example, "o" command opens an url in current tab. You should type "on" command to open a url in new tab.It's like using VIM, necessary to pressing ESC button before typing a ffuzbl command.

You can find the ffuzbl's commands and some essential Firefox shortcuts below.

FFUZBL Commands

Press ESC button, type your FFUZBL command and press enter to execute it.Here is the list that contains default commands of FFUZBL;
  • oOpen given url in current tab
  • onOpen given url in new tab
  • cCopy URL of current tab to clipboard
  • prefOpen Preferences Window
  • evalExecute given Javascript code
  • fbToggle Firebug Window
  • aboutOpen About Window of Firefox
  • ffuzblOpen FFUZBL About Page
PS: Try Ubiquity addon with FFUZBL.
